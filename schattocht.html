Plantage Schattocht

Mobile application

'Plantage schattocht' was my first school project. As a group of 5 students we were assigned the task to create an interactive tour for an area in Amsterdam for a specific target group. In our case, parents with children in the 'Plantagebuurt'.

My job in this project was the development of the application. It was one of my first things I built with Javascript. I tried to give the application a 'pirate' feeling and made the map look like a real treasure map. So I added custom markers for the points of interests and a pirate to show the current location. Doing so taught me a lot about Javascript, API's and real time events like geolocation.
