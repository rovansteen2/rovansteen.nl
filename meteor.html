Meteor Blog

Javascript application

For a school assignment I got the opportunity to work with an amazing new web framework built on top of Node: Meteor. It's an amazing package to create powerful, reactive web applications that is fundamentally different than we are used to right now. No need to refresh for updates or wait for rerendered pages. With Meteor you can access the web in a way that feels completely new and exciting. I too was really amazed by this framework and immediately wanted to built applications with it. So in consultation with my teacher I decided to start building a blog system that uses Meteor. After a few months of developing and learning a lot about this whole new approach I completed this blog system that's responsive and reactive and feels really different than things I've made in the past.
